## Instalación

1. Clonar este repositorio:
   ```bash
   git clone https://github.com/PiaZaniboni/zaniboni-backend-77525
   cd zaniboni-backend-77525
   ```

2. Instalar las dependencias del proyecto:
   ```bash
   npm i
   ```

3. Run codigo:
   ```bash
   npm run dev
   ```
